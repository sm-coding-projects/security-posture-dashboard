# Security Posture Dashboard

A comprehensive Next.js dashboard for monitoring and managing your organization's security posture, compliance, and risk assessment.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Custom error pages for 404 and general errors
- Professional UI matching the app design system
- Helpful navigation and user guidance
